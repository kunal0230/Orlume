Contribute to Orlume

Help us build the future of open-source photo editing.

Everyone Can Contribute

Whether you're a developer, designer, writer, or just passionate about photography — there's a place for you in the Orlume community. Every contribution matters, no matter how small.

View on GitHub

Ways to Contribute

Choose how you'd like to help make Orlume better.

Code

Fix bugs, add features, optimize performance. Check our GitHub issues for "good first issue" labels.

Documentation

Improve our docs, write tutorials, or help translate Orlume to other languages.

Bug Reports

Found something broken? Open a detailed issue on GitHub to help us fix it.

Feature Ideas

Have a great idea? Share it in our discussions or submit a feature request.

Design

UI/UX improvements, icons, graphics — help make Orlume more beautiful.

Feedback

Test new features, share your experience, help us understand what users need.

Our Tech Stack

JavaScript / ES6+ WebGL2 / WebGPU TensorFlow.js ONNX Runtime Vite GLSL Shaders
