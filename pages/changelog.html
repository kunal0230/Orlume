Changelog

What's New

A complete history of updates, features, and improvements to Orlume.

v1.5.0 January 2, 2026

Professional Presets & HSL Improvements

  • New Professional Presets library with 40+ cinematic looks across 6 categories
  • New Preset intensity slider for blending effects
  • New Colorful gradient backgrounds for HSL hue sliders
  • Improved HSL color mixer with smoother transitions
  • Improved Overall UI polish and micro-animations
  • Fixed Preset reset button now properly clears all adjustments
v1.4.0 December 28, 2025

Advanced 3D Model Export

  • New Export depth maps as 3D OBJ meshes with textures
  • New Interactive 3D preview with orbit controls
  • New Configurable mesh resolution and depth scale
  • Improved Depth estimation accuracy for complex scenes
  • Fixed Shadow slider now works correctly in relight mode
v1.3.0 December 20, 2025

8-Channel HSL Color Mixer

  • New Complete HSL color mixer with 8 color channels
  • New Independent Hue, Saturation, and Luminance controls per channel
  • New Collapsible channel panels for cleaner workflow
  • Improved GPU shader performance for real-time HSL adjustments
  • Fixed Global undo/redo now works across all editing modes
v1.2.0 December 15, 2025

Enhanced Relighting System

  • New Interactive light source positioning with visual indicator
  • New Multiple light types: Key, Fill, Rim, and Ambient
  • New Light color picker for creative effects
  • Improved Faster depth estimation with optimized models
  • Improved More realistic shadow falloff calculation
v1.1.0 December 10, 2025

Background Removal & Effects

  • New AI-powered background removal with edge refinement
  • New Vignette effect with customizable intensity and radius
  • New Film grain effect for analog aesthetics
  • Improved Export quality up to 4K resolution
  • Fixed Image loading issues on Safari browsers
v1.0.0 December 1, 2025

Initial Release

  • New 3D Relighting with AI depth estimation
  • New Basic adjustments: Exposure, Contrast, Highlights, Shadows
  • New Color controls: Temperature, Tint, Vibrance, Saturation
  • New Crop and rotation tools
  • New Export to JPEG, PNG, and WebP formats
  • New WebGL2 GPU acceleration for real-time preview
